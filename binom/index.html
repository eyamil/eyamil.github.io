The Binomial Theorem

How it's used, where it comes from, and other interesting rambling

Notation

The binomial theorem is insanely useful for doing things, but we have to clear up some notation before we can actually start discussing it. You might also know it as the "Choose" function, and for a very good reason, but we'll get to that later.
There are two ways to express "n Choose k". First, there's straight-up choose notation, \( \mathrm{ }_{n}C_k \). The second way to write the same statement is \( \binom{n}{k} \). On this blog you'll probably see it expressed via the second kind of notation.
